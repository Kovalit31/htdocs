Installing Apache

Apache is a webserver, which you can download and install on many platforms.
Apache is opensource software, so you cn watch out a source code of Apache.

To install it, you need to go on this website: here.

Install


Linux


For Ubuntu/Debian:
sudo apt-get install apache2 php
For Fedora/RedHat:
sudo dnf install httpd -y

Windows


For Windows Apache can be found here.

Configuring Apache for Windows


Unpack Apache to folder, where isn't whitespace (such as C:\Apache) and
unpack php to folder, where isn't whitespace (such as C:\PHP).
Then download mod_fcgid from Apche Lounge website.
Unpack file "mod_fcgid.so" to folder Apache\modules
Then go to Apache\conf, open file httpd.conf and write this in the start of the file (This line must be after comments (comments starts with #)):

Define SRVROOT = "path/to/apache"

Then write this in the end of the file:

Define PHPPATH = "path/to/php"
LoadModule fcgid_module modules/mod_fcgid.so
<IfModule fcgid_module>


FcgidMaxProcesses 300
FcgidMaxProcessesPerClass 300

FcgidOutputBufferSize 65536
FcgidConnectTimeout 10
FcgidProcessLifeTime 0
FcgidMaxRequestsPerProcess 0
FcgidMinProcessesPerClass 0
FcgidFixPathinfo 0
FcgidProcessLifeTime 0
FcgidZombieScanInterval 20
FcgidMaxRequestLen 536870912
FcgidIOTimeout 120
FcgidTimeScore 3

FcgidPassHeader Authorization

FcgidInitialEnv PHPRC "${PHPPATH}"
FcgidInitialEnv PATH "${PHPPATH}";C:\\WINDOWS\\system32;C:\\WINDOWS;C:\\WINDOWS\\System32\\Wbem;"
FcgidInitialEnv SystemRoot "C:\\Windows"
FcgidInitialEnv SystemDrive "C:"
FcgidInitialEnv TEMP "C:\\WINDOWS\\TEMP"
FcgidInitialEnv TMP "C:\\WINDOWS\\TEMP"
FcgidInitialEnv windir "C:\\WINDOWS"
<Files ~ "\.php$">
Options Indexes FollowSymLinks ExecCGI
AddHandler fcgid-script .php
FcgidWrapper "${PHPPATH}\\php-cgi.exe" .php
</Files>
</IfModule>


If you want allow cookies (for this site this must be enabled), find parameter "#LoadModule session_cookie_module modules/mod_session_cookie.so" and
enable it, deleting "#"
